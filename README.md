# End-to-End Cybersecurity Incident Case Study

This repository documents a guided, hands-on cybersecurity project focused on analyzing a simulated cyber incident from detection through risk assessment.

## Project Goals
- Understand how cybersecurity incidents are detected and analyzed
- Practice log analysis, threat modeling, and incident response
- Explore multiple areas of cybersecurity including SOC operations, DFIR, threat intelligence, and risk management
